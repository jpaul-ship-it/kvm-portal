AI Quote Assistant — API Key

Set your Anthropic API key to enable AI auto-fill in the Sales quote builder. The key is stored as a server environment variable on Render — never exposed to the browser.

To set the API key: Go to your Render dashboard → your kvm-portal service → Environment → add variable ANTHROPIC_API_KEY with your key value (starts with sk-ant-...). Then redeploy.

Skills Master List

These skills appear as checkboxes on employee profiles and on project required-skills selection. Skills are grouped by category. Inactive skills are hidden from selection but preserved for historical data.

Trucks / Crews

Each truck is a crew anchored by a lead technician. Row names on the schedule will display the lead's name. Use Shop / Flex / Delivery row types for non-crew rows. Temporary crews only appear on the schedule during their date range.

Default GL Accounts per Bill Category

When a bill line doesn't have a specific GL account assigned, these defaults are used. Type an account name that matches exactly what's in your QuickBooks file (e.g., "5100 · COGS — Materials").

Chart of Accounts

Your QuickBooks Chart of Accounts. Used when coding bill lines to GL accounts for the QB export. Upload the accounts once from QB, and the portal will stay in sync with it.

Download Database Backup

Download the full SQLite database file. Keep a copy before every major deploy. The file is approximately the size of all your users, customers, quotes, projects, timeclock entries and settings combined.

🧪 Test Data

Seed the database with realistic test customers, sites, contacts, quotes, and projects so you can exercise workflows without entering real data. Every test record is tagged and can be removed with one click when you’re ready. All test records have names prefixed with “(TEST)” for easy identification.
